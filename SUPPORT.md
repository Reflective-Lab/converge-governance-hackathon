# Support

Use GitHub issues for bugs, tasks, and product requests that can be discussed publicly.

Do not use public issues for vulnerabilities or secrets. Report security issues through [SECURITY.md](SECURITY.md).

For local development problems, include:

- Operating system and version
- Rust version
- Bun version
- Command that failed
- Relevant logs with secrets removed
- Whether the problem affects backend, web, desktop, or cloud deployment
